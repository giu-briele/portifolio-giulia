Cardápio Digital

Desenvolvi um aplicativo de cardápio digital interativo utilizando React Native, atuando desde a prototipação até a entrega final. Estruturei o layout em seções organizadas, com exibição em grid responsivo (três itens por linha) e implementei um modal interativo para detalhamento de cada item selecionado. Utilizei componentes reutilizáveis para garantir escalabilidade e mantenibilidade do código.

Tecnologias e ferramentas:

  • React Native – desenvolvimento da interface mobile e navegação
  • JavaScript (ES6+) – lógica de estados e interações com o usuário
  • React Navigation – gerenciamento de rotas entre telas e navegação entre seções
  • Flexbox – estruturação em grade responsiva
  • StyleSheet – estilização customizada para manter a identidade visual
