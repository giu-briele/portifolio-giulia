Emprestai

Sistema de gerenciamento de biblioteca com funcionalidades para cadastro de usuários, livros e controle de empréstimos. O objetivo foi proporcionar uma interface intuitiva para facilitar a administração da biblioteca de forma organizada e acessível.

Neste projeto, atuei exclusivamente no desenvolvimento do front-end, criando toda identidade visual, prototipação e no desenvolvimento criando componentes reutilizáveis com React e utilizando useState e useEffect para o gerenciamento de estado e efeitos colaterais nas interfaces. Também desenvolvi formulários modais, listas dinâmicas e interações completas entre os elementos da interface.

Tecnologias e ferramentas utilizadas:

  • React – criação da interface e componentes funcionais
  • JavaScript (ES6+) – manipulação de estados e eventos
  • useState, useEffect – controle de formulários, listas e atualizações de UI
  • CSS – estilização das páginas com foco em responsividade
