Nova 4E

Sistema integrado desenvolvido para o site de uma ONG, com o objetivo de facilitar a interação entre a organização e seus voluntários, além de aprimorar a gestão de eventos e o recebimento de feedbacks. O sistema busca oferecer uma interface amigável e intuitiva para os usuários, ao mesmo tempo em que disponibiliza ferramentas eficazes para a equipe da ONG gerenciar suas atividades de forma mais eficiente.

Neste projeto, atuei integralmente no front-end, desde a prototipação até o desenvolvimento da interface.

Tecnologias e ferramentas utilizadas:

  • Figma – prototipação de todas as páginas
  • HTML, CSS, Bootstrap – desenvolvimento da interface web e navegação
  • JavaScript – lógica de estados e interações com o usuário
  • PHP – integração com o back-end e manipulação de dados
  • phpMyAdmin – gerenciamento do banco de dados MySQL
