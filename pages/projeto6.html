Receitas Sazonais

Desenvolvi um aplicativo de receitas temáticas com foco em estações do ano e feriados comemorativos, utilizando React Native. Atuei desde a concepção do layout até a implementação das funcionalidades principais. Organizei o conteúdo em categorias sazonais, com navegação intuitiva e visual atraente, usando imagens e cores temáticas para cada época. Criei componentes reutilizáveis para cards de receitas, com detalhamento individual de ingredientes e modo de preparo.

Tecnologias e ferramentas:

  • React Native – criação da interface multiplataforma (Android, iOS e Web)
  • JavaScript (ES6+) – controle da navegação, filtragem de dados e interatividade
  • React Navigation – gerenciamento de rotas entre telas e navegação entre seções
  • FlatList – renderização otimizada de listas de receitas
  • StyleSheet – aplicação de estilos temáticos para identidade visual sazonal
